# Trelo Labs VDC Document Intelligence Production Deployment Plan\n\n**Document Curiosity Brain Decision**\n**Date:** 2026-04-24\n**Status:** Approved for Production Deployment\n\n## Overview\nThis plan deploys the Document Curiosity Brain (VDC Document Intelligence app) to trelolabs.com publicly. Assumes Ubuntu 22.04+ server, Node.js app running on localhost:3000 (or similar), PM2 for process management.\n\n**Server Prerequisites:**\n- Ubuntu server with non-root sudo user\n- Domain: trelolabs.com (A record to server IP)\n- Node.js app deployed and tested locally (e.g., via PM2: `pm2 start app.js --name vdc`)\n- Firewall: UFW enabled\n\n## 1. Server Hardening & Base Setup\n```bash\n# Update system\nsudo apt update && sudo apt upgrade -y\n\n# Install essentials\nsudo apt install -y nginx certbot python3-certbot-nginx ufw fail2ban build-essential\n\n# Create app user (optional, for isolation)\nsudo adduser --system --group --no-create-home vdcapp\n# Set app dir ownership: sudo chown -R vdcapp:vdcapp /path/to/app\n```\n\n## 2. Firewall Rules (UFW)\n```bash\nsudo ufw allow OpenSSH\nsudo ufw allow 'Nginx Full'  # Ports 80/443\nsudo ufw --force enable\nsudo ufw status\n```\n**Rules:**\n- Allow: SSH (22), HTTP (80), HTTPS (443)\n- Deny: All else\n\n## 3. Nginx Configuration\nCreate `/etc/nginx/sites-available/trelolabs.com`:\n```nginx\nserver {\n    listen 80;\n    server_name trelolabs.com www.trelolabs.com;\n    # Redirect HTTP to HTTPS\n    return 301 https://$server_name$request_uri;\n}\n\nserver {\n    listen 443 ssl http2;\n    server_name trelolabs.com www.trelolabs.com;\n\n    # SSL (auto-configured by Certbot)\n    ssl_certificate /etc/letsencrypt/live/trelolabs.com/fullchain.pem;\n    ssl_certificate_key /etc/letsencrypt/live/trelolabs.com/privkey.pem;\n    include /etc/letsencrypt/options-ssl-nginx.conf;\n    ssl_dhparam /etc/letsencrypt/ssl-dhparams.pem;\n\n    # Security Headers\n    add_header X-Frame-Options \"SAMEORIGIN\" always;\n    add_header X-XSS-Protection \"1; mode=block\" always;\n    add_header X-Content-Type-Options \"nosniff\" always;\n    add_header Referrer-Policy \"no-referrer-when-downgrade\" always;\n    add_header Content-Security-Policy \"default-src 'self'; script-src 'self' 'unsafe-inline' 'unsafe-eval'; style-src 'self' 'unsafe-inline'; img-src 'self' data: https:; font-src 'self' data: https:; connect-src 'self'; frame-ancestors 'self';\" always;\n    add_header Strict-Transport-Security \"max-age=63072000; includeSubDomains; preload\" always;\n\n    # Rate Limiting (10 req/min per IP, burst 20)\n    limit_req_zone $binary_remote_addr zone@vdc:10m rate=10r/m;\n    limit_req zone@vdc burst=20 nodelay;\n\n    # File Upload Limits (for document processing: 100MB max)\n    client_max_body_size 100M;\n    proxy_request_buffering off;\n\n    # Compression\n    gzip on;\n    gzip_vary on;\n    gzip_min_length 1024;\n    gzip_proxied any;\n    gzip_comp_level 6;\n    gzip_types text/plain text/css text/xml text/javascript application/javascript application/xml+rss application/json;\n\n    root /var/www/trelolabs.com/html;  # Static files\n    index index.html;\n\n    # Reverse proxy to Node.js app\n    location / {\n        proxy_pass http://localhost:3000;\n        proxy_http_version 1.1;\n        proxy_set_header Upgrade $http_upgrade;\n        proxy_set_header Connection 'upgrade';\n        proxy_set_header Host $host;\n        proxy_set_header X-Real-IP $remote_addr;\n        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;\n        proxy_set_header X-Forwarded-Proto $scheme;\n        proxy_cache_bypass $http_upgrade;\n    }\n\n    # API endpoints (e.g., document upload)\n    location /api/ {\n        proxy_pass http://localhost:3000;\n        limit_req zone@vdc burst=5 nodelay;  # Stricter for API\n        proxy_buffering off;\n    }\n\n    # Static assets\n    location /static/ {\n        alias /path/to/app/static/;\n        expires 1y;\n        add_header Cache-Control \"public, immutable\";\n    }\n}\n```\nEnable site:\n```bash\nsudo ln -s /etc/nginx/sites-available/trelolabs.com /etc/nginx/sites-enabled/\nsudo rm /etc/nginx/sites-enabled/default\nsudo nginx -t && sudo systemctl reload nginx\n```\n\n## 4. SSL Setup (Let's Encrypt)\n```bash\nsudo certbot --nginx -d trelolabs.com -d www.trelolabs.com\n# Choose redirect option (2)\n```\n**Renewal:** Auto via systemd timer (test: `sudo certbot renew --dry-run`)\n\n## 5. Additional Security\n### Fail2Ban\n```bash\nsudo nano /etc/fail2ban/jail.local\n# Add under [DEFAULT]:\nbantime = 1h\nfindtime = 10m\nmaxretry = 5\n\n# Nginx jails\n[nginx-http-auth]\nenabled = true\n\n[nginx-limit-req]\nenabled = true\nfilter = nginx-limit-req\nlogpath = /var/log/nginx/error.log\n\nsudo systemctl enable fail2ban && sudo systemctl start fail2ban\n```\n\n### Node.js App Security\n- Bind to `localhost:3000`\n- Use `helmet` middleware for headers\n- Validate/sanitize uploads (e.g., `multer` limits: {fileSize: 100*1024*1024})\n- Rate limit in app (e.g., `express-rate-limit`)\n- Env vars for secrets\n\n## 6. Integration Testing Steps\n1. **Basic Connectivity:** `curl -I https://trelolabs.com` (200 OK, HSTS header)\n2. **SSL Check:** https://www.ssllabs.com/ssltest/ (A+ grade)\n3. **Security Headers:** `curl -I https://trelolabs.com` (check CSP, etc.)\n4. **Upload Test:** Upload sample docs via app UI/API, verify processing\n5. **Rate Limit:** `ab -n 50 -c 1 https://trelolabs.com/api/upload` (429 after limit)\n6. **Load Test:** `hey -n 1000 -c 50 https://trelolabs.com`\n7. **Fail2Ban:** Simulate bad auth, check `fail2ban-client status nginx-http-auth`\n8. **PM2 Health:** `pm2 monit`, ensure app restarts on kill\n9. **Logs:** `tail -f /var/log/nginx/access.log /var/log/nginx/error.log`\n10. **Downtime Test:** Restart services, verify quick recovery\n\n## 7. Monitoring & Maintenance\n- PM2: `pm2 monit`, logs\n- Nginx: `/var/log/nginx/*.log`\n- Certbot: `certbot renew --dry-run` (cron)\n- UFW/Fail2Ban: `fail2ban-client status`\n- Updates: Weekly `apt upgrade`, `npm update`\n\n## Rollback\n- `sudo certbot delete`\n- `sudo rm /etc/nginx/sites-enabled/trelolabs.com`\n- `sudo systemctl reload nginx`\n\n**Go Live Checklist:**\n- [ ] DNS propagated\n- [ ] All tests pass\n- [ ] Backup server snapshot\n- [ ] Announce deployment
